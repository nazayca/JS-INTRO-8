TERNARY OPERATOR

Find max number between 2 numbers 
let num1 = 20, num2 = 10;

let max;

if(num1 > num2) {
    max = num1;
}
else {
    max = num2;
}

We can simplify above if-else using ternary operator 
let num1 = 20, num2 = 10;
let max = num1 > num2 ? num1 : num2;
